    There are three ways to implement CSS styles to our HTML page:

  • Inline CSS: An internal style sheet may be used if one single HTML page has a unique style. The internal style is defined inside the "style" tag element, inside the head section.
  • Outline CSS: An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.
  • External CSS: you can change the look of an entire website by changing just one file. You have just to include a reference to the external style sheet file inside the "link" tag element, inside the head section.

    • This whole div element, containing the h2, the list and this line elements, is styled with External CSS, see the style.css file
